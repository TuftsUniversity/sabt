Submission Agreement XML Schemas

Currently, there are two discrete schemas for submission agreements. Regular Submission Agreements and Standing Submission Agreements are just enough different from each other to warrant having their definitions stored in two separate files.

Schema document locations

The canonical schema documents live in TAPER's Subversion repository, in the schema/ directory. They are both W3 XML Schema documents.

  • ssa.xsd - the schema for SSAs
  • rsa.xsd - the schema for RSAs
  • sa_common.xsd - a schema containing definitions common to both RSAs and SSAs, used by both main schemas

The files are meant to be self-documenting, and follow the standards set down by "Element List--Submission Agreements 2.0".
